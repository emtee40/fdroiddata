Summary:Network Mapper is Android frontend for Nmap
Description:
Frontend will help you to download and install Nmap as well as use it.
Nmap will help you to discover hosts, protocols, open ports, services and 
its configuration and vulnerabilities on networks.

Frontend supports all known Android architectures: arm, mips and x86.

Nmap binaries are transferred using HTTPS by default.

Network Mapper is released under open source license and source is available 
on GitHub:
[https://github.com/kost/NetworkMapper]

Android binary versions of Nmap are built using following port:
[https://github.com/kost/nmap-android]
