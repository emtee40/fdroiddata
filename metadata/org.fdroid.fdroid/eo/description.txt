F-Droid estas instalebla katalogo de Libera kaj Malfermkoda Programaro (FOSS) por operaciumo Android. La kliento plifaciligas foliumadon, instaladon kaj spuradon de ĝisdatigoj ĉe via aparato.

Ĝi konektas al ĉiuj F-Droid-kongruaj deponejoj. La implicita deponejo estas gastigata ĉe f-droid.org kaj enhavas nur vere Liberan kaj Malfermkodan Programaron.

Android per si mem estas malferma tielmaniere, ke vi senlime povas instali aplikaĵoj el kie ajn vi volas, tamen ekzistas bonaj kialoj por uzi F-Droid kiel via administrilo de liberaj programoj:
* esti sciigita kiam ĝisdatigoj disponeblas
* (malnepre) elŝuti kaj instali ĝisdatigojn aŭtomate 
* esplori pri malpli novaj kaj nestabilaj versioj 
* filtri aplikaĵojn, kiuj ne kongruas kun via aparato 
* esplori aplikaĵojn per kategorioj kaj serĉeblaj priskriboj 
* aliri kunigitajn ligilojn por donacoj, fontkodo, ktp.
* esti sekura per kontroli signaturojn kaj kontrolsumojn (haketaĵojn) de apk-oj
